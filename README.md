## Instalação

## Requisitos

PHP 8.1.1

## Realizar o clone do projeto
git clone https://github.com/adrianomsx8/123milhas-teste

## Baixar as depencias
$ cd 123milhas-teste

$ composer install

## Iniciar o servidor

$ php artisa serve

## Acessar via navegador

http://localhost:8000/api
